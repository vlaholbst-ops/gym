# Зал — программа A/B

Мобильная страница-компаньон: https://vlaholbst-ops.github.io/gym/

## Как устроено

**Источник правды по весам — журнал тренировок в базе знаний**, а не этот репозиторий:
`~/Knowledge/Здоровье/Владимир/Журнал тренировок/ГГГГ-ММ-ДД — День X.md`

| Файл | Что в нём |
|---|---|
| `program.json` | Упражнения, техника, запреты, шаг прогрессии. **Весов здесь нет.** |
| `template.html` | Вёрстка страницы. Плейсхолдеры `{{DAYS}}`, `{{WARMUP}}`, `{{PHOTO*}}`. |
| `build.mjs` | Читает журнал → считает прогрессию → собирает `index.html`. |
| `index.html` | **Генерируется. Руками не править.** |

## Цикл после тренировки

```bash
# 1. Занести тренировку в базу
node ~/.claude/tools/knowledge/health.mjs workout --day B \
     --exercise "Жим ногами, стопы выше" --sets "62кг×12, 62кг×12, 62кг×11"

# 2. Пересобрать и опубликовать — веса подтянутся сами
cd ~/projects/gym && node build.mjs
git add -A && git commit -m "тренировка 13.07" && git push
```

**Веса руками нигде не вписывать.** Прогрессия считается: все подходы в верхней границе
повторов → +шаг; недотянул до нижней → держим; скакал по весу → выравниваем на максимальном.
