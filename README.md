# Зал — программа A/B

Мобильная страница-компаньон для тренировок: программа, фото тренажёров, прошлые веса,
таймер отдыха, запись подходов.

**Не редактировать `index.html` руками** — он собирается.

```bash
# правишь программу и веса «в прошлый раз» в source.html (блок DAYS),
# затем пересобираешь и пушишь:
node build.mjs && cp gym-built.html index.html && git add -A && git commit -m "..." && git push
```

Источник правды по тренировкам — база знаний `~/Knowledge/Здоровье`.
Эта страница — её витрина, а не второе хранилище.
